# Тестовое задание

## Installation


1. Клонировать
   ```sh
   git clone https://github.com/Nikolay-Twin/test_work_symfony.git
   ```
2. Инсталировать (Docker)
   ```sh
   ./tools/app.sh init
   ```
3. Переименовать .env.dev в .env

4. Прописать в hosts:
   ```sh
   127.0.0.1       srt.local
   ```
5. Посмотреть)  http://srt.local


## Usage

1 Веб
